실제 화면 맞춤형 상세 버전

초등학생도 무조건 성공하는
웹사이트 완벽 배포 가이드

보이는 그대로 따라 클릭만 하세요!

🔥
PART 1

파이어베이스 (DB 만들기)

1

메뉴 찾기

파이어베이스 왼쪽 메뉴 목록에서 [데이터베이스 및 스토리지] 글자를 클릭하세요. 아래로 메뉴가 열리면 [Firestore]를 클릭합니다.

※ 현재 고객님 화면(image_093d6d)과 일치하는 단계입니다.

2

데이터베이스 만들기

검은색 배경 화면 가운데에 나타난 주황색 테두리의 데이터베이스 만들기 버튼을 클릭하세요.

  • 위치는 기본값(미국 등)으로 두고 '다음' 클릭
  • '프로덕션 모드'로 두고 '만들기' 클릭
!

데이터베이스 문 열어주기 (매우 중요)

데이터베이스가 만들어지면, 위쪽 탭에서 [규칙(Rules)]을 클릭합니다. 아래 네모 박스처럼 영어 단어 하나를 고친 후 꼭 [게시(Publish)] 버튼을 누르세요.

allow read, write: if false true;
3

내 컴퓨터와 연결할 '열쇠 코드' 복사하기

  • 왼쪽 맨 위 톱니바퀴 아이콘(⚙️)을 누르고 [프로젝트 설정] 클릭!
  • 화면을 맨 아래로 내려서 </> 모양 아이콘 (웹) 클릭!
  • 앱 닉네임을 아무거나(예: web) 적고 [앱 등록] 클릭!
  • 화면에 나타나는 const firebaseConfig = { ... } 모양의 코드를 전부 마우스로 드래그해서 복사(Ctrl+C) 합니다.
💻
PART 2

내 HTML 파일 수정

복사한 열쇠 붙여넣기

내 컴퓨터(바탕화면 등)에 있는 엑셀용 HTML 파일을 메모장이나 코드 편집기(VS Code)로 엽니다.

아까 파이어베이스에서 복사해 둔 열쇠 코드를 HTML 파일 안의 알맞은 곳에 붙여넣기(Ctrl+V) 하고 반드시 저장(Ctrl+S) 합니다.

🐙
PART 3

깃허브 (인터넷 주소 만들기)

1

새 저장소 만들기

깃허브 사이트에서 초록색 [New] 버튼을 누릅니다. 저장소 이름(예: receiving)을 영어로 적고, 반드시 Public (공개)을 선택한 후 바닥의 초록색 [Create repository] 버튼을 누릅니다.

2

파일 올리는 곳으로 이동

새 저장소가 만들어지면 여러 영어 글씨가 나옵니다. 파란색 박스 부분의 글씨들 중에서 uploading an existing file 이라는 파란색 글씨를 찾아 클릭하세요.

... or uploading an existing file ...

※ 현재 고객님 화면(image_093e7e)에서 바로 클릭해야 할 버튼입니다!

3

내 HTML 파일 올리기

파일 업로드 화면이 나오면, 아까 열쇠 코드를 넣고 저장했던 내 컴퓨터의 HTML 파일을 마우스로 끌어다가 이 창에 놓습니다. (드래그 앤 드롭)

파일 이름이 목록에 나타나면 밑에 있는 초록색 [Commit changes] 버튼을 꾹 누릅니다.

4

🎉 마지막! 진짜 인터넷 주소 켜기

  • 위쪽 탭 메뉴 중 맨 끝에 있는 톱니바퀴 [Settings] 클릭!
  • 왼쪽 메뉴 목록에서 [Pages] 클릭!
  • 화면 가운데 'Branch' 밑에 있는 None ▼ 버튼을 눌러서 main (또는 master)으로 바꿉니다.
  • 옆에 생기는 [Save] 버튼 클릭!
